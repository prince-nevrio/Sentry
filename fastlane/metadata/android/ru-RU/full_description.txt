Мини приложение для применения дополнительных политик безопасности.

Оно может:
* ограничить максимальное количество неудачных попыток ввода пароля
* отключить передачу данных по USB (Android 12, USB HAL 1.3, Device Owner)

Также Вы можете выдать разрешение на чтение уведомлений устройства для отключения передачи данных
по USB автоматически при выключении экрана.

Разрешения:
* DEVICE_ADMIN - ограничить максимальное количество неудачных попыток ввода пароля
* DEVICE_OWNER - отключить передачу данных по USB
* NOTIFICATION_LISTENER - получать события блокировки (опционально)

Это свободное программное обеспечение с открытым исходным кодом.
